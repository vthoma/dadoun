# Loterie

**Simple lottery app**

## Requirements
* PHP 8.1 or higher
* Composer
* NPM

## Usage
* Clone this repo

* Open a terminal and run the following commands
```
composer install 
php artisan migrate
php artisan serve
```

* Open a new terminal and run the following commands
```
npm install
npm run dev
```

* Open your browser and go to http://localhost:8000
